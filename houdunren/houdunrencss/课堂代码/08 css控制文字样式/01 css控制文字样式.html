后盾网 人人做后盾

网易娱乐专稿9月14日报道 昨晚，郭德纲晒出一张大儿子郭麒麟为自己捶背的照片，两人都身穿白色上衣，郭德纲手拿蒲扇坐在椅子上，儿子坐在他身后认真地给老爸捶背，郭德纲眉眼弯弯，露出幸福笑容。他开心写道：“门前有马非为富，家中有人不算穷。” 网友们纷纷表示“难得看到郭老板和少帮主同框”“国产亲子装”“写得真好 且画面温馨感人”。网易娱乐专稿9月14日报道 昨晚，郭德纲晒出一张大儿子郭麒麟为自己捶背的照片，两人都身穿白色上衣，郭德纲手拿蒲扇坐在椅子上，儿子坐在他身后认真地给老爸捶背，郭德纲眉眼弯弯，露出幸福笑容。他开心写道：“门前有马非为富，家中有人不算穷。” 网友们纷纷表示“难得看到郭老板和少帮主同框”“国产亲子装”“写得真好 且画面温馨感人”。网易娱乐专稿9月14日报道 昨晚，郭德纲晒出一张大儿子郭麒麟为自己捶背的照片，两人都身穿白色上衣，郭德纲手拿蒲扇坐在椅子上，儿子坐在他身后认真地给老爸捶背，郭德纲眉眼弯弯，露出幸福笑容。他开心写道：“门前有马非为富，家中有人不算穷。” 网友们纷纷表示“难得看到郭老板和少帮主同框”“国产亲子装”“写得真好 且画面温馨感人”。
